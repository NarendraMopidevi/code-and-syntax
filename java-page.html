Introduction to Java

Java is a powerful, high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle Corporation) in 1995. It is designed to be platform-independent and follows the principle of "Write Once, Run Anywhere" (WORA), meaning code written in Java can run on any system that has a Java Virtual Machine (JVM).

How Java Works

Java programs are first written in source files with a .java extension. These files are compiled by the Java Compiler (javac) into platform-independent bytecode stored in .class files. The JVM then executes this bytecode by converting it into machine code.

Compilation Flow 
Source Code (.java) → Compiler (javac) → Bytecode (.class) → JVM → Machine Code

Key Features of Java

  • Platform Independent: Runs on any OS with a JVM.
  • Object-Oriented: Based on classes and objects.
  • Secure: Strong memory management, sandbox execution.
  • Robust: Strong error and exception handling.
  • Multithreaded: Supports concurrent task execution.
  • High Performance: Uses Just-In-Time (JIT) compilation.
  • Portable: Consistent behavior across systems.

Basic Syntax

Every Java program begins with a class definition and a main() method, which serves as the entry point of execution. The JVM starts running your program from main().

Example Program

Java 
// Simple Java Program
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Understanding the main() Method

The line public static void main(String[] args) is the JVM's entry point. Here's what each keyword means:

  • public: Makes main() accessible to the JVM from outside the class.
  • static: Belongs to the class itself — JVM calls it without creating an object.
  • void: Does not return any value.
  • main: The predefined method name recognized by JVM as the starting point.
  • String[] args: Stores command-line arguments passed at runtime.

Command-Line Arguments

Java 
class Example {
    public static void main(String[] args) {
        if (args.length > 0) {
            System.out.println("First argument: " + args[0]);
        } else {
            System.out.println("No arguments were passed!");
        }
    }
}

Output

First argument: Hello

Internal Execution Flow

  1. JVM starts and loads the class into memory.
  2. It searches for public static void main(String[] args).
  3. It creates a String array and fills it with command-line arguments.
  4. It executes the statements inside main().
  5. After completion, the JVM terminates.

Java Data Types

In Java, data types define the kind of values a variable can store and the memory it occupies. Java is a strongly typed language — every variable must have a declared type before use.

Types of Data Types

  • Primitive Data Types: Predefined by Java, used for simple values like numbers and characters.
  • Non-Primitive Data Types: Created by programmers — include classes, arrays, and interfaces.

Primitive Data Types

Example Code

Java 
class DataTypeExample {
    public static void main(String[] args) {
        int age = 21;
        double salary = 35000.50;
        char grade = 'A';
        boolean isStudent = true;

        System.out.println("Age: " + age);
        System.out.println("Salary: " + salary);
        System.out.println("Grade: " + grade);
        System.out.println("Is Student: " + isStudent);
    }
}

Output

Age: 21
Salary: 35000.5
Grade: A
Is Student: true

Key Points

  • Primitive types are stored by value and have fixed sizes.
  • Non-primitive types (like Strings) store references, not direct values.
  • Always choose the smallest data type that fits your value to optimize memory.

Java Control Statements

Control statements determine the flow of program execution — making decisions, repeating tasks, and controlling which code executes based on conditions.

Types of Control Statements

  • Decision-making: if, if-else, nested if, switch
  • Looping: for, while, do-while
  • Jump: break, continue, return

Decision-Making Statements

1. if Statement

Executes a block of code only when the given condition is true.

Java 
class IfExample {
    public static void main(String[] args) {
        int age = 20;
        if (age >= 18) {
            System.out.println("Eligible to vote");
        }
    }
}
Eligible to vote

2. if-else Statement

Provides an alternative block when the condition is false.

Java 
class IfElseExample {
    public static void main(String[] args) {
        int num = -10;
        if (num >= 0) {
            System.out.println("Positive Number");
        } else {
            System.out.println("Negative Number");
        }
    }
}
Negative Number

3. Nested if Statement

An if inside another if, used for multiple conditional checks.

Java 
class NestedIfExample {
    public static void main(String[] args) {
        int age = 25;
        String nationality = "Indian";
        if (nationality.equals("Indian")) {
            if (age >= 18) {
                System.out.println("Eligible to Vote in India");
            }
        }
    }
}
Eligible to Vote in India

4. switch Statement

Executes one case from multiple options based on a variable's value.

Java 
class SwitchExample {
    public static void main(String[] args) {
        int day = 3;
        switch (day) {
            case 1: System.out.println("Monday"); break;
            case 2: System.out.println("Tuesday"); break;
            case 3: System.out.println("Wednesday"); break;
            default: System.out.println("Invalid Day");
        }
    }
}
Wednesday

Looping Statements

1. for Loop

Executes a block a specific number of times.

Java 
class ForLoopExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Count: " + i);
        }
    }
}
Count: 1
Count: 2
Count: 3

2. while Loop

Executes repeatedly as long as the condition is true.

Java 
class WhileExample {
    public static void main(String[] args) {
        int i = 1;
        while (i <= 3) {
            System.out.println("Iteration: " + i);
            i++;
        }
    }
}
Iteration: 1
Iteration: 2
Iteration: 3

3. do-while Loop

Executes the block at least once before checking the condition.

Java 
class DoWhileExample {
    public static void main(String[] args) {
        int i = 1;
        do {
            System.out.println("Value: " + i);
            i++;
        } while (i <= 3);
    }
}
Value: 1
Value: 2
Value: 3

Jump Statements

1. break Statement

Exits from a loop or switch block before it naturally ends.

Java 
class BreakExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            if (i == 3) break;
            System.out.println("i = " + i);
        }
    }
}
i = 1
i = 2

2. continue Statement

Skips the current iteration and continues with the next loop cycle.

Java 
class ContinueExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            if (i == 3) continue;
            System.out.println("i = " + i);
        }
    }
}
i = 1
i = 2
i = 4
i = 5

3. return Statement

Exits a method and optionally returns a value to the caller.

Java 
class ReturnExample {
    public static void main(String[] args) {
        int result = addNumbers(5, 10);
        System.out.println("Sum: " + result);
    }

    static int addNumbers(int a, int b) {
        return a + b;
    }
}
Sum: 15

Summary

  • if, if-else, and switch handle decision-making.
  • for, while, and do-while handle repetitive tasks.
  • break, continue, and return control flow within loops and methods.

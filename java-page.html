Introduction to Java

Java is a high-level, object-oriented programming language developed by Sun Microsystems. It is designed to be platform-independent and follows the “Write Once, Run Anywhere” principle.

Basic Syntax

Every Java program begins with a class definition and a main() method, which serves as the entry point.


// Simple Java Program
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

OOP Concepts

Java supports the main OOP principles: Encapsulation, Inheritance, Polymorphism, and Abstraction.

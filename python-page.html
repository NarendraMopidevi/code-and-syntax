Introduction to Python

Python is a high-level, interpreted, and dynamically typed programming language known for its simplicity and versatility. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Basic Syntax

Python code is executed line by line (interpreted). Unlike Java or C++, indentation determines code blocks instead of curly braces.


# Simple Python Program
print("Hello, World!")

# Indentation is important
if True:
    print("This is indented properly!")

Data Types

Python supports several built-in data types such as int, float, str, list, tuple, dict, and set.


# Examples of data types
name = "Alice"      # str
age = 25            # int
pi = 3.14           # float
numbers = [1, 2, 3] # list
person = {"name": "Alice", "age": 25} # dict

Loops & Control Flow

Python provides for and while loops to iterate over sequences.


# For loop example
for i in range(5):
    print("Iteration:", i)

# While loop example
count = 0
while count < 3:
    print("Count:", count)
    count += 1

Functions

Functions in Python are defined using the def keyword. They help in reusability and modular coding.


def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))

Object-Oriented Programming

Python supports classes and objects for OOP. You can create classes using the class keyword.


class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print(f"Hello, I'm {self.name} and I'm {self.age} years old.")

p1 = Person("Bob", 30)
p1.greet()

Modules & Packages

Modules in Python are files containing code. You can import them using the import statement.


# math module example
import math

print(math.sqrt(16))

Exception Handling

Python handles runtime errors using try, except, and finally blocks.


try:
    num = int(input("Enter a number: "))
    print(10 / num)
except ZeroDivisionError:
    print("Cannot divide by zero!")
except ValueError:
    print("Invalid input.")
finally:
    print("Execution complete.")

File Handling

Python provides simple functions for file operations like reading and writing.


# Writing to a file
with open("example.txt", "w") as file:
    file.write("Hello, Python!")

# Reading from a file
with open("example.txt", "r") as file:
    print(file.read())

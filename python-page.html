Introduction to Python

Python is a high-level, interpreted, and dynamically typed programming language known for its simplicity and versatility. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Basic Syntax

Python code is executed line by line. Unlike Java or C++, indentation determines code blocks instead of curly braces.

Python 
print("Hello, World!")

if True:
    print("This is indented properly!")

Data Types

Python supports several built-in data types: int, float, str, list, tuple, dict, and set.

Python 
name = "Alice"       # str
age = 25             # int
pi = 3.14            # float
numbers = [1, 2, 3]  # list
person = {"name": "Alice", "age": 25}  # dict

Loops & Control Flow

Python provides for and while loops to iterate over sequences.

Python 
# For loop
for i in range(5):
    print("Iteration:", i)

# While loop
count = 0
while count < 3:
    print("Count:", count)
    count += 1

Functions

Functions in Python are defined using the def keyword. They enable reusability and modular coding.

Python 
def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))

Output

Hello, Alice!

Object-Oriented Programming

Python supports classes and objects. You can create classes using the class keyword.

Python 
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print(f"Hello, I'm {self.name} and I'm {self.age} years old.")

p1 = Person("Bob", 30)
p1.greet()

Output

Hello, I'm Bob and I'm 30 years old.

Modules & Packages

Modules are files containing Python code. You import them using the import statement.

Python 
import math

print(math.sqrt(16))  # Output: 4.0

Exception Handling

Python handles runtime errors using try, except, and finally blocks.

Python 
try:
    num = int(input("Enter a number: "))
    print(10 / num)
except ZeroDivisionError:
    print("Cannot divide by zero!")
except ValueError:
    print("Invalid input.")
finally:
    print("Execution complete.")

File Handling

Python provides simple built-in functions for file operations.

Python 
# Writing to a file
with open("example.txt", "w") as file:
    file.write("Hello, Python!")

# Reading from a file
with open("example.txt", "r") as file:
    print(file.read())

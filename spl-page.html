Introduction to SQL

SQL (Structured Query Language) is the standard language used to communicate with relational databases. It allows you to create, read, update, and delete data stored in tables.

Detailed content coming soon.

SELECT Statement

The SELECT statement retrieves data from one or more tables.

SQL 
SELECT name, age FROM students;

-- Select all columns
SELECT * FROM students;

WHERE Clause

The WHERE clause filters records based on a condition.

SQL 
SELECT * FROM students WHERE age > 18;

SELECT * FROM students WHERE name = 'Alice';

JOINs

JOINs combine rows from two or more tables based on a related column.

SQL 
-- INNER JOIN
SELECT students.name, courses.course_name
FROM students
INNER JOIN enrollments ON students.id = enrollments.student_id
INNER JOIN courses ON enrollments.course_id = courses.id;

Aggregate Functions

Aggregate functions perform calculations on a set of values and return a single result.

SQL 
SELECT COUNT(*) FROM students;
SELECT AVG(age) FROM students;
SELECT MAX(score), MIN(score) FROM results;
SELECT SUM(salary) FROM employees;

GROUP BY & HAVING

GROUP BY groups rows sharing a value. HAVING filters those groups.

SQL 
SELECT department, COUNT(*) AS total
FROM employees
GROUP BY department
HAVING COUNT(*) > 5;

Subqueries

A subquery is a query nested inside another query.

SQL 
SELECT name FROM employees
WHERE salary > (SELECT AVG(salary) FROM employees);

DDL Commands

Data Definition Language commands define the database structure.

SQL 
-- Create a table
CREATE TABLE students (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT
);

-- Alter a table
ALTER TABLE students ADD email VARCHAR(100);

-- Drop a table
DROP TABLE students;
